## 🚀 Установка

1. Клонуйте репозиторій:
```bash
git clone https://github.com/Stas5252s/coffee-shop-website.git
cd coffee-shop-website
```

2. Відкрийте `index.html` у браузері

Або використовуйте Live Server для локальної розробки
