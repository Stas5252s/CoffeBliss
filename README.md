# ☕ BrewBliss - Премиальная кофейня

Современный и стильный сайт для кофейни с минималистичным дизайном и интерактивными элементами.

## 🎨 Особенности

✅ **Современный дизайн** - Минималистичный стиль с уютной кофейной палитрой  
✅ **Адаптивный дизайн** - Работает идеально на мобильных, планшетах и десктопе  
✅ **Интерактивные элементы** - Плавная прокрутка, hover-эффекты и анимации  
✅ **Быстрая загрузка** - Оптимизировано для высокой производительности  
✅ **SEO-оптимизировано** - Семантическая разметка и хорошая структура  

## 🎯 Разделы сайта

### 1. **Главная (Hero)**
- Крупный логотип BrewBliss
- Привлекающий слоган и кол-во к действию (CTA)
- SVG иллюстрация кофе с анимацией

### 2. **Меню**
- ☕ **Кофейные напитки** (6 видов)
  - Эспрессо, Капучино, Латте, Мокка, Американо, Ледяной Латте
- 🍰 **Десерты и закуски** (6 видов)
  - Круассаны, Брауни, Тарты, Чизкейк, Маффины, Панна-котта
- Красивые карточки с описанием и ценами
- Интерактивные hover-эффекты

### 3. **О нас**
- История кофейни
- Философия работы
- Описание атмосферы
- Статистика (1500+ клиентов, 50+ сортов кофе, 4.9★ рейтинг)

### 4. **Контакты**
- Адрес, телефон, email
- Интерактивная карта Google Maps
- Социальные сети (Instagram, Facebook, VK, Twitter)

## 🚀 Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Stas5252s/coffee-shop-website.git
cd coffee-shop-website
```

2. Откройте `index.html` в браузере

Или используйте Live Server для локальной разработки
